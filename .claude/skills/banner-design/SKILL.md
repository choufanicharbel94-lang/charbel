# Banner Design - Multi-Format Creative Banner System

This skill enables creation of banners across social media, advertising, web, and print platforms using AI-powered design generation.

## Core Capabilities

The system handles "banners for social media, ads, website heroes, creative assets, and print" with multiple art direction options per request. It explicitly does NOT cover video editing, full website design, or print production.

## Key Workflow Phases

**Requirements Gathering**: Collect details on purpose, platform/dimensions, content, brand guidelines, style preference, and desired quantity through structured questions.

**Research & Art Direction**: Leverage the `ui-ux-pro-max` skill and Pinterest research to establish 2-3 complementary design directions using established reference materials.

**Design Generation**: Create HTML/CSS banners with exact platform dimensions, then generate visual elements using `ai-artist` and `ai-multimodal` skills, selecting between standard (2K, fast) and pro (4K, detailed) image models based on complexity.

**Export to Images**: Convert HTML designs to PNG using Chrome DevTools at specified dimensions with automatic compression for files exceeding 5MB.

**Presentation & Iteration**: Display exported options with design rationale and dimensions, then refine based on user feedback.

## Platform Size Reference

Common dimensions include: Facebook cover (820×312), Twitter header (1500×500), YouTube channel art (2560×1440), Instagram story (1080×1920), and website hero (1920×600-1080).

## Design Standards

Implementation requires "max 2 typefaces, single CTA, 4.5:1 contrast ratio" with critical content positioned within central 70-80% of the canvas. Ad text should remain under 20% of total area.
