# Brand Agent Documentation

This resource provides a comprehensive toolkit for managing brand identity, voice, messaging, and visual standards.

## Core Capabilities

The brand agent handles "Brand identity, voice, messaging, asset management, and consistency frameworks." Key use cases include voice definition, visual standards development, messaging creation, and brand audits.

## Essential Tools

Four primary scripts facilitate brand management:

- **inject-brand-context.cjs** – Extracts brand context for prompt injection
- **sync-brand-to-tokens.cjs** – Synchronizes guidelines to design tokens
- **validate-asset.cjs** – Checks asset naming, size, and format compliance
- **extract-colors.cjs** – Analyzes and compares colors against defined palettes

## Workflow Process

The brand sync workflow follows this sequence: edit brand guidelines, sync to design tokens, then verify outputs. The source of truth resides in `docs/brand-guidelines.md`, with synchronized outputs in both JSON and CSS formats.

## Reference Materials

Comprehensive guides cover voice frameworks, visual identity standards, messaging structures, consistency checklists, asset organization, color management, typography specs, logo usage rules, and approval workflows—all available in the references directory.
