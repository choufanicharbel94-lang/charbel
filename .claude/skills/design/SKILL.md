# Design Skill Overview

This is a comprehensive design skill for Claude agents, licensed under MIT by claudekit (v2.1.0).

## Core Capabilities

The skill covers unified design work including brand identity, design tokens, UI styling, logo creation, corporate identity programs (CIP), presentations, banners, social media imagery, and icon design across multiple platforms (Facebook, Twitter, LinkedIn, YouTube, Instagram, Pinterest, TikTok, Threads, Google Ads).

## Key Features

**Logo Design:** Offers 55+ styles, 30 color palettes, and 25 industry guides using Gemini AI models. Generation includes white background output with optional HTML preview galleries.

**CIP (Corporate Identity):** Provides 50+ deliverables across 20 styles and industries, supporting mockup generation with or without logos, including HTML presentation rendering.

**Banners:** Supports 22 art direction styles for social, ads, web, and print platforms with specific dimension guidelines and design rules.

**Social Photos:** Enables multi-platform image design using HTML/CSS converted to screenshots, with workflow guidance for concept ideation through final export.

**Icons:** Generates 15 style variations across 12 categories using Gemini 3.1 Pro, outputting SVG text with batch and multi-size export options.

## Technical Requirements

Python 3 is required (Windows users substitute `python` for `python3`). Setup involves setting the GEMINI_API_KEY environment variable and installing google-genai and pillow packages.
