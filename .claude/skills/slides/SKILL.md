# Slides: Strategic HTML Presentation Design

This tool creates data-driven presentations using HTML, Chart.js, and design principles optimized for marketing and pitch decks.

## Key Capabilities

The slides command supports "strategic HTML presentation design with data visualization." It's designed for scenarios including marketing presentations, pitch decks, and content that benefits from charts and responsive layouts.

## How It Works

The system operates through a routing mechanism: it parses your input, identifies the subcommand (starting with `create`), then loads relevant reference materials to guide generation. The knowledge base includes resources on layout patterns, copywriting formulas, and strategic slide structures.

## Available Resources

The tool references four main guides:
- Layout patterns for slide organization
- HTML template standards
- Copywriting formulas for persuasive messaging
- Strategic approaches to slide sequencing

## Usage Pattern

Invoke with format: `[topic] [slide-count]` — for example, "SaaS pricing strategy 8" would generate eight slides focused on that subject.

This approach combines visual design principles with persuasive writing techniques to create presentations that communicate effectively.
